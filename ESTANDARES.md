# Estándares Internacionales y Normativas Aplicables
## Operaciones de Perforación, Pulling y Workover

Este documento detalla el marco normativo bajo el cual se desarrollan los procedimientos operativos y los módulos de simulación técnica de este repositorio.

---

## 1. Estructuras y Equipos de Torre
El diseño y mantenimiento de las unidades de servicio de pozos (Pulling/Workover) se rigen por:

* **API Spec 4F**: Especificación para estructuras de perforación y servicio de pozos. Define las capacidades de carga del mástil y la subestructura.
* **API RP 4G**: Prácticas recomendadas para la inspección y mantenimiento de estructuras. *Es la base para el Check-list de montaje incluido en este repositorio.*

## 2. Sistema de Izaje y Cables de Acero
Para garantizar la seguridad en maniobras de pesca y extracción de columnas:

* **API Spec 9A**: Especificación para cables de acero. Regula el cable de 1" (tambor principal) y 9/16" (tambor de pistoneo).
* **API RP 9B**: Aplicación, cuidado y uso de cables de acero en servicio de pozos.
* **API Spec 8C**: Requisitos para equipos de izaje (elevadores, ganchos y aparejos) con el fin de evitar fallas por fatiga de materiales.

## 3. Sistema de Bombeo Mecánico y Varillas
Las operaciones de pesca de varillas y vástagos (Clase 11) siguen los estándares:

* **API Spec 11B**: Especificación para varillas de bombeo, vástagos pulidos y acoplamientos.
* **API RP 11BR**: Prácticas recomendadas para el manejo y cuidado de varillas. Establece los límites de torque para evitar desenrosques no deseados.
* **API Spec 11E**: Especificación para unidades de bombeo (A.I.B.), fundamental para el re-enganche del pozo tras la intervención.

## 4. Control de Pozos y Seguridad (Well Control)
Normas críticas para los programas de "Ahogo de Pozo" y "Descompresión":

* **API Standard 53**: Sistemas de equipos de prevención de reventones (BOP). Define las pruebas de hermeticidad y presión para la BOP de varillas y de tubing.
* **API RP 59**: Prácticas recomendadas para el control de pozos, asegurando la integridad del yacimiento durante la intervención.

---

## 5. Clasificación de Equipos según Capacidad
De acuerdo a los estándares industriales, diferenciamos las unidades de servicio por su capacidad de tracción y torque:

| Tipo de Equipo | Norma de Diseño | Capacidad Nominal Sugerida |
| :--- | :--- | :--- |
| **Pulling** | API 4F | Hasta 100,000 lbs (45,000 kg) |
| **Workover** | API 4F / 7K | > 150,000 lbs con Subestructura |
