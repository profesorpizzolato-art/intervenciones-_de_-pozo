# Marco Normativo y Estándares API

Para garantizar la integridad mecánica y la seguridad operativa en las intervenciones de Pulling y Workover, todas las operaciones descritas en este repositorio se rigen bajo las siguientes normas del **American Petroleum Institute (API)**:

## 1. Equipos de Torre y Estructuras
* **API Spec 4F**: Especificación para Estructuras de Perforación y Servicio de Pozos. Regula el diseño, fabricación y límites de carga de los mástiles de los equipos de Pulling y Workover.
* **API RP 4G**: Prácticas recomendadas para la inspección, mantenimiento y reparación de estructuras de torre. (Crucial para el cumplimiento del *Check-list* de montaje).

## 2. Elementos de Izaje y Cables
* **API Spec 9A**: Especificación para cables de acero (Wire Rope). Define las propiedades del cable de 1" para el tambor principal y 9/16" para el tambor de pistoneo.
* **API RP 9B**: Prácticas recomendadas para la aplicación, cuidado y uso de cables de acero en servicio de pozos.
* **API Spec 8C**: Especificación para equipos de perforación y elevación de producción (Ganchos, elevadores, eslabones y aparejos).

## 3. Barras de Bombeo y Pesca (Sucker Rods)
* **API Spec 11B**: Especificación para Varillas de Bombeo, vástagos y acoplamientos. Define las roscas y metalurgia para evitar fallas por fatiga.
* **API RP 11BR**: Prácticas recomendadas para el cuidado y manipulación de las varillas de bombeo (Fundamental para las maniobras de extracción en "tiros dobles" mencionadas en la Clase 11).

## 4. Control de Pozos (Well Control)
* **API Standard 53**: Sistemas de equipos de prevención de reventones (BOP) para operaciones de perforación y servicios.
* **API RP 59**: Prácticas recomendadas para el control de pozos (Procedimientos de ahogo y descompresión a pileta detallados en los programas de pesca).

## 5. Equipos de Bombeo Mecánico (A.I.B.)
* **API Spec 11E**: Especificación para Unidades de Bombeo. Regula la geometría y límites de carga de la "Cabeza de Mula" y el equipo de superficie.

---

> **Nota Técnica:** El cumplimiento de estas normas es obligatorio para asegurar que la frecuencia de intervención se mantenga en el objetivo de **una cada 2 años**, minimizando los costos operativos que pueden alcanzar el 30% del total en yacimientos profundos.
