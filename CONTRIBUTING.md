# Guía de Contribución - Material Técnico MENFA

¡Gracias por el interés en mejorar este recurso educativo! Para mantener la calidad técnica y pedagógica de los simuladores y manuales, pedimos seguir estas pautas:

## 🛠 ¿Cómo puedo ayudar?
1. **Correcciones Técnicas**: Si encuentras un error en una fórmula de hidráulica, torque o en los pasos de un programa de pozo (Workstep).
2. **Nuevos Módulos**: Si tienes sugerencias para el simulador de Perforación o el de Producción.
3. **Documentación**: Mejorar las explicaciones de las clases para que sean más claras para los alumnos.

## 📝 Formato de Archivos
* **Manuales**: Deben estar en formato Markdown (`.md`).
* **Código**: Si envías scripts de Python/Streamlit, asegúrate de que estén comentados siguiendo las normas PEP 8.
* **Normativas**: Cualquier cambio debe citar la norma API correspondiente (ej. API Spec 4F).

## 🚀 Proceso de Envío
1. Haz un **Fork** del repositorio.
2. Crea una **Rama (Branch)** con el nombre de la mejora (ej: `fix-calculo-caudal`).
3. Realiza tus cambios y haz un **Commit** descriptivo.
4. Envía un **Pull Request** para que Fabricio pueda revisar los cambios antes de integrarlos.
